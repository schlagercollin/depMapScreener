Description

An R-Shiny application to visualize virtual screens using the Broad Institute's DepMap data.

  1. Parameters (start here)

    2. Define two groups of cell lines to compare genetic dependencies.

    "Knock Out" separates cell lines into two groups determined by whether they have damaging mutations in a given gene.

  2. Screen Plot

    3. View differential dependencies for each gene.

    A volcano plot with difference in mean depedency score (x-axis) and significance of the difference (y-axis) is plotted for every gene. Negative x-axis values imply enhanced dependency in the condition group.

  3. Gene Plot

    4. View the depedency score distributions across cell lines for a given gene.

    Violin plots displaying the distribution of dependency score for the condition group and the control group.

  4. Table

    5. View and download the quantitative results of the screen.

    Statistics on the differential dependencies for each gene across the two cell line groups.

  5. Conditioned Cell Lines

    6. View the cell lines that belong to your condition group.

    The cell lines that have been grouped together based on the parameters chosen.

  6. Mutation Lookup

    7. Explore the mutation space represented in the DepMap data gene-by-gene.

    Given an input gene, displays the cell lines that have annotated mutations in that gene.
