Description

An R-Shiny application to visualize perform virtual screen analysis using the Broad Institute's DepMap data.

  1. Parameters (start here)

    2. Define two groups of cell lines to compare genetic dependencies.

    • "Knock Out" separates cell lines into two groups determined by whether they have damaging mutations in a given gene.

    • "Expression" separates cell lines into groups based on their expression values. This is loosely analogous to a virtual reporter screen, where highly fluorescent cells are compared against a baseline group.

    • "Lineage" separates cell lines based on their original lineage.

    • "Custom" allows you to specify a list of DepMap cell line IDs (e.g. ACH-000009) that you want to include in the condition group. The rest of the cell lines profiled will be used as the control group.

  2. Enrichment Results

    3. View differential dependencies for each gene. View and download the quantitative results of the screen.

    A volcano plot with difference in mean depedency score (x-axis) and significance of the difference (y-axis) is plotted for every gene. Negative x-axis values imply enhanced dependency in the condition group. The table on the differential dependencies for each gene across the two cell line groups. This output comes from the R limma package and its use here is adapted from Chan et al (see below).

  3. Gene-Level

    4. View the depedency score distributions across cell lines for a given gene.

    Violin plots displaying the distribution of dependency score for the condition group and the control group.

  4. View Cell Lines

    5. View the cell lines that belong to your condition and control groups.

    The cell lines have been grouped together based on the parameters chosen.

  5. Mutation Lookup

    6. Explore the mutation space represented in the DepMap data by gene or by cell type.

    Given an input gene, displays the cell lines that have annotated mutations in that gene.
